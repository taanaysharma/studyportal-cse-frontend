Course Information - Algorithm Design & Analysis

Code: CSE 221
Type: Core
Semester: IV

Prerequisites

Data Structure Basics of Computer Programming
Course Outcomes ▼
  1. Analyze time and space complexity for the given algorithms.
  2. Use various searching, sorting and graph traversal algorithms.
  3. Use various techniques for efficient algorithm design like divide and conquer, greedy and dynamic algorithms.
  4. Describe minimum spanning tree algorithms, Single source shortest path, all pair shortest path algorithms in a graph and backtracking concepts.
  5. Describe polynomial and non-polynomial problems, classify NP hard and NP complete problems with their application.
Course Content ▼

1. Fundamentals of Algorithm

Analysis of complexity, recurrence relations, disjoint set structure.

2. Algorithm Design Techniques

Algorithm design techniques and Masters Theorem to compute algorithms time complexity.

3. Core Algorithmic Paradigms

Greedy algorithm and Dynamic programming concepts.

4. Graph Algorithms

Spanning Tree algorithms, graph shortest path algorithms.

5. Advanced Traversal & Backtracking

Branch and Bound techniques, Backtracking algorithms and graph traversal algorithms (BFS and DFS).

6. Computational Complexity

Introduction of NP-completeness and NP-hardness problems.

Course Information - Computer System Organization

Code: CSE 222 (CSE 24222)
Type: Core
Semester: IV

Prerequisites

Digital Circuit Design (CSE-214)
Course Outcomes ▼
  1. Understand the physical organization of digital computers, including their registers, data flow, microoperations, and control functions.
  2. Use various instructions from basic computers to write assembly-language instructions for different data processing tasks.
  3. Compare the performance of digital computers with varying architectural choices and evaluate using AMDHAL's Law.
  4. Comprehend the organization and implementation of computer memories.
  5. Gain knowledge of the methods used by digital computers for interfacing with a variety of input/output devices.
Course Content ▼

1. Basic Computer Organization and Central Processor

Overview of basic computer organization and design, Central processor organizations: basic building blocks, bus organized computer memory, address structure, register transfer languages, instruction formats, expanding op-codes and addressing modes.

2. Control Unit Organization

Hardwired control & microprogrammed control organization, control memory, address sequencing micro-instruction formats, micro-program sequencer, and micro-programming.

3. Arithmetic Processor Design & Pipelining

Addition and subtraction algorithm, multiplication algorithm, division algorithm, processor configuration, and floating-point arithmetic. Performance evaluation of processors. Introduction to pipeline processing, pipeline hazards.

4. Input-Output Organization

Asynchronous Data Transfer, Asynchronous Communication Interface, Modes of Transfer: Interrupt-Initiated, Direct Memory Access (DMA).

5. Memory Organization

Main Memory, Auxiliary Memory, Associative Memory: Hardware Organization, Cache Memory: Mapping Schemes.

Reference Books

Course Information - Software Engineering

Code: CSE 24221
Type: Core
Semester: IV

Prerequisites

Basic Programming Concepts
Course Content Overview ▼

This course covers the systematic approach to the development, operation, and maintenance of software. It spans SDLC phases, process models (Waterfall, Agile, Spiral), and essential management topics such as Risk Management, Quality Assurance, and ISO/CMM standards.

Key technical areas include Software Project Management (Scheduling, Estimation, COCOMO/Putnam Models, Halstead metrics), structural Software Design methodologies, Coding practices, and rigorous Software Testing techniques. The course also details contemporary adoptions like Low-code/No-code, Cloud Computing, Progressive Web Apps, and the integration of AI/Machine Learning in Software Engineering.

Course Information - Data Communication

Code: CSE 224
Type: Core
Semester: IV

Prerequisites

Basic understanding of computer networks
Course Content Overview ▼

This course provides a comprehensive foundation in data communications. Topics range from layered communication architecture to advanced physical layer phenomena like signal sampling, channel bandwidth, transmission impairments (attenuation, distortion, noise), and the channel capacity of noisy/noise-free channels.

It also extensively covers transmission media (guided and unguided), line coding schemes, modulation techniques, and multiplexing schemas. Beyond the physical properties, the course explores modern communication needs, diving into error detection/correction, satellite communications, multimedia network performance, audio/video compression, and basics of network cryptography.

Course Information - Fundamentals of Entrepreneurship

Code: HUM 251
Type: Core
Semester: III/IV

Prerequisites

Basic understanding of organizational functioning
Course Outcomes ▼
  1. Course will inculcate basic knowledge of Industrial Laws & basics of management principles for engineering students to handle all the matters related to industrial day to day functions and technicalities.
  2. Engineering students will be in position to take appropriate decision to understand the industrial Laws and application of administrative principles for smooth functioning of Industry and a fresh idea to setup new business.
  3. Boost up the capacity of engineers to develop Business ideas to become successful entrepreneurs to setup new operational business units to address the need of society at national and local level.
Course Content ▼

1. Introduction to Entrepreneurship

Basics of Entrepreneurship: Meaning, Nature, Scope, importance and present scenario, Evolution of Entrepreneurship, Factors influencing Entrepreneurship. Characteristics of an Entrepreneur, Types of Entrepreneur. New Generation: social entrepreneurship, tourism, women entrepreneurship. Barriers to Entrepreneurship.

2. Rural Entrepreneurship

Rural entrepreneurship in India, Meaning, Concept, Needs, Problems, Rural Industrialization, Entrepreneurial development in backward areas, Problems and Risk in entrepreneurship.

3. Industrial Laws

Factory Act 1948, Minimum wages Act, Provident Fund Act, Dispute Act, Contract Labor Act, Compensation Act, Inter State Migrant worker Act, Sexual Harassment Act, SEZ, National Small Industries, Quality Standard (ISO), SIDBI, New Pension Act 2004.

4. Entrepreneurial Motivation

Meaning and Relevance of Motivation, Maslow's Theory, Herzberg's Theory, Douglas McGregor. Organizational Development Theories. Fredric Winslow Taylor, Marry Parker Follett, Elton Mayo, Max Weber, Henry Fayol, Power, Accountability, Responsibility, Control, Transparency and Conflict Resolutions.

5. Government Initiatives

ATAL Innovation Mission, SAMRIDH Scheme, Multiplier Grant Scheme (MGS), Start-up India Seed Fund, Single Point Registration Scheme, Stand-up India Scheme, Ministry of Skill Development and Entrepreneurship, Case studies in software Technology Parks.
